1. ウィンドウの作成
===================

ウィンドウの作成は以下のようにして行うことができます.

.. code:: python

   import pgutil

   # Playgroundインスタンスの作成
   pg = pgutil.Playground()

   # メインループ
   # 1フレームに1回実行される
   # フレームレートは60fpsに設定されているため1秒間に60回以上実行されることはない
   while pg.run():
       # 何もしない
       pass


pgutilでは上のプログラムに示すような

1. PlaygroundまたはGLgroundオブジェクトのインスタンス化

2. メインループ

の2つを含むプログラムを基本としてこれに機能を追加する形で可視化を行います.
