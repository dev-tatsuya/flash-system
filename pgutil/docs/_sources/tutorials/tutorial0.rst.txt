0. はじめに
===========

.. _header-n37:

pgutilとは
----------

pgutilは様々な種類のデータを最低限のコードで高速,
インタラクティブに可視化することに重点を置いたデータ可視化ライブラリです.

pgutilはPlaygroundとGLgroundという2つのクラスからなり,
用途に応じてこの2つのクラスのうちどちらかを使用することになります.

Playgroundはpythonのゲーム作成用ライブラリpygameをベースにしたクラスで主に2次元の描画に特化したクラスになります.
このクラスでは2次元データの可視化や基本的な図形の描画,
ピクセル単位でのウィンドウの表示内容の操作などに便利な様々な関数を使用することができます.

GLgroundはpygameとオープンソースの3D描画ライブラリOpenGLをベースにしたクラスで主に3次元の描画に特化したクラスになります.
このクラスでは3次元データの可視化や2次元データの3次元空間上での可視化,
大量の頂点からなる複雑な図形の描画などに便利な様々な関数を提供しています.

また,
PlaygroundとGLgroundの両方で定義済みの関数を特定のキーやマウスのボタンに関連付けるといった形でインタラクティブなプログラムを最低限のコードで作成できる仕組みを利用することができます.

.. _header-n44:

インストール
------------

pgutilは現在一般公開されていないため,
pipを使用してインストールすることはできません. このため,
pgutilのソースコードが格納されたフォルダ(基本的にpgutilという名前です.)をpythonがモジュールをインポート可能な位置に置くことでインストールを行います.
もしもpythonの開発にPyCharmを使用しているのであればプロジェクトフォルダにpgutilのソースコードが格納されたフォルダを置くのが良いでしょう.

また, pgutilは外部ライブラリのpygameとPyOpenGLに依存しているため,
この2つのライブラリを別途インストールする必要があります.
これらのライブラリのインストールはpip経由で行うことができるためAnaconda
Promptを開いて

.. code:: 

   pip install pygame

と

.. code:: 

   pip install PyOpenGL

のコマンドを入力することによってインストールが可能です.
